= sinatra-partial-layout

examples of partials and layouts on Sinatra.  the idea is keep things DRY.

== Requirements

* gem install sinatra
* gem install sinatra-contrib
* gem install haml

== See the output with

* ruby sinatra_partial_layout.rb
* browse to http://localhost:4567/
